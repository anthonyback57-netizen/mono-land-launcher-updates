Publication GitHub Pages

1) Cree un repo public: mono-land-launcher-updates
2) Upload tout le contenu de ce dossier a la racine du repo
3) Active GitHub Pages (Deploy from branch -> main / root)
4) URL manifest final:
   https://<ton-user>.github.io/mono-land-launcher-updates/manifest.json
5) Ouvre manifest.json et remplace REPLACE_ME par ton user GitHub

Format du manifest:
{
  "version": "1.0.2",
  "installerUrl": "https://<ton-user>.github.io/mono-land-launcher-updates/releases/Mono-Land%20Launcher%20Setup%201.0.2.exe"
}
